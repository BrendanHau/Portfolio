Maplestory Checklist
A website designed around the MMORPG video game "Maplestory". This site provides the user with a simple checklist interface where they can keep track of their progress within the game.
The site utilizes the browser's local storage feature to save information pertaining to the current state of the checklist. The checklist will automatically reset certain categories at specific times (indicated by the countdowns) removing the need to manually reset the items. In the case that the user does not have the browser opened once the reset occurs, info about the last time visited is also stored. On site load, a function runs to check if the last time visited overlaps one of the reset timers. If it does, it will reset the corresponding category.
